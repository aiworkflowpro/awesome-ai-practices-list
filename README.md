## Why This List

Most "awesome AI" lists are static link dumps that go stale in weeks. This one is different:

- **Auto-curated** — DigestOps scans X every 6 hours for practitioner-shared workflows
- **No hot takes** — every entry is something someone actually built, with reusable takeaways
- **Source-linked** — every practice traces back to the original post with full attribution

> [!NOTE]
> This list focuses on **what practitioners ship**, not what they say. If someone built it, tested it, and shared the result — it belongs here.
